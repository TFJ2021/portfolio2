## <img src="https://cdn.jsdelivr.net/gh/Readme-Workflows/Readme-Icons@main/icons/octicons/PullRequestDrafted.svg"> Getting Started

### Prerequisites

* Node.js (>= 18 recommended)
* npm

### Installation

```bash
# clone the repository
git clone https://github.com/TFJ2021/portfolio2.git

# go into the project directory
cd portfolio-v2
```

### Development

```bash
# start simple http server
npx http-server
```

The app will be available at `http://localhost:8080` by default.
